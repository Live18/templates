My First Email Template. What do you think?
I used a lesson written by Nicole Merlin as a tutorial for this project. It was written in 2013 so some ideas and standards may be updated with new ideas. As a first attempt, I'm quite proud of myself. Feel free to take this code and use it. You can DM me on LinkedIn for the repo link.
The most important lesson I picked up from this project was that email structures aren't normalized with anything like Bootstrap or other structures similar to web design. This means everything on an email must be in a table. Additionally, the various email clients have their own structures and standards.
 
Part of the lesson was using a validator from W3 Schools. I ran this code thru it and came up with several errors and warnings I'm not able to fix at this time. It's an ongoing learning process for me. This first attempt was a solid one, and I know I can gett better including adding GIFs and other animations.
